## üßê About <a name = "about"></a>

you can track down their habit by clicking the days that the task was completed.

clicking a day along side the name of the task, change the default value of false to true, which will be marked in blue.

track multiple habits by adding more tasks in the bottom of the tracker.

- newHabit is stored as an object with two kye values of "habit" and "daysComplete". newHabit objects are stored in an array named listArr

- number of dyas are shown according to the displayed month
